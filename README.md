# 中国城市天气地图

一个美观简洁的网页应用，展示中国主要城市的天气信息。

## 功能特点

- 显示中国地图和主要城市位置
- 点击城市查看实时天气信息
- 响应式设计，适配各种设备
- 简洁美观的用户界面

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- Leaflet.js 地图库
- OpenWeatherMap API (模拟数据)

## 使用方法

1. 克隆仓库到本地
2. 在浏览器中打开 `index.html` 文件
3. 点击地图上的城市标记查看天气信息

## 注意事项

当前版本使用模拟数据。如需使用真实天气数据，请在 `app.js` 中替换 OpenWeatherMap API 密钥，并取消注释相关代码。
